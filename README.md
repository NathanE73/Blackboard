## Requirements

- iOS 8.0+
- Xcode 7.3+

## Installation

### Downloading

```bash
$ curl https://raw.githubusercontent.com/NathanE73/Blackboard/master/bin/blackboard > blackboard
```

### Making It Executable

```bash
$ chmod +x blackboard
```
